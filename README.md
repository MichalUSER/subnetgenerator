# Subnet Generator

**Site to generate subnets easily**

## Get started

1. Clone the repo

```
git clone https://github.com/MichalUSER/subnetgenerator.git
```

2. Install dependencies

```
npm install
# or
pnpm install
```

3. Run dev server

```
npm run dev
# or
pnpm run dev
```

## Tech stack

- Svelte
- Typescript
- Tailwind
