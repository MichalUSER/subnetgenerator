# Subnet Generator

**Site to generate subnets easily**

## Developing

1. Clone the repo

```
git clone https://github.com/MichalUSER/vlsmcalc.git
```

2. Install dependencies

```
npm install
# or
pnpm install
```

3. Run dev server

```
npm run dev
# or
pnpm run dev
```

## Building

```bash
npm run build
# or
pnpm build
```

## Tech stack

- Svelte
- TypeScript
- Tailwind

> [Check it out](https://vlsmcalc.vercel.app/)
